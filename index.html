Tutorial de Next.js

Asignatura: Desarrollo de Aplicaciones en Red

Autor:Roberto Fraire

1. Justificación del Framework

Next.js es un framework basado en React que permite crear aplicaciones web rápidas y modernas. Facilita el desarrollo de sitios estáticos y dinámicos con renderizado en el servidor (SSR), mejorando la velocidad y el SEO.

¿Para qué sirve?

  • Desarrollar sitios con SEO optimizado
  • Crear aplicaciones web completas
  • Renderizado en servidor y cliente

Lenguajes y prerrequisitos

Usa JavaScript o TypeScript, junto con HTML y CSS. Requiere tener instalado Node.js y conocimientos básicos de React.

2. Instalación

Para comenzar con Next.js se debe instalar Node.js. Luego, desde la terminal:

npx create-next-app@latest mi-proyecto
cd mi-proyecto
npm run dev

Esto creará y ejecutará un servidor local en http://localhost:3000.

3. Primeros Pasos: "Hola Mundo"

En el archivo pages/index.js escribe:

export default function Home() {
  return (
    <div>
      <h1>¡Hola Mundo desde Next.js!</h1>
    </div>
  );
}

Guarda y visualiza el resultado en el navegador.

4. Utilización del Framework

Rutas automáticas

En Next.js cada archivo en pages/ se convierte en una ruta. Por ejemplo:

  • pages/index.js → /
  • pages/about.js → /about

Componentes

export default function Header() {
  return (<h2>Mi sitio en Next.js</h2>);
}

Importa el componente en tu página principal para reutilizarlo.

5. Funcionamiento Interno

Next.js organiza las páginas dentro de la carpeta pages/ y gestiona automáticamente las rutas.

Ejemplo de API interna

export default function handler(req, res) {
  res.status(200).json({ mensaje: 'Hola desde la API de Next.js' });
}

Guarda este archivo como pages/api/hola.js y accede desde el navegador.

6. Conclusiones

  • Next.js amplía las capacidades de React para proyectos reales.
  • Permite combinar frontend y backend en un mismo entorno.
  • Ideal para sitios modernos, rápidos y optimizados.
